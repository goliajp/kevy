## §1 Atomic memory orderings used

kevy-rt's cross-shard signaling uses:
- `parked[i]: AtomicBool` (sender side: load Acquire to check; receiver side: store Release)
- `pending_wakes[i]: bool` (single-shard local; not atomic — only mutated by the owning shard)

Documented in shard.rs runtime loop comments. Reviewed.
